# Real-Time B2B Fraud Detection Platform

## Overview

A backend-based fraud detection platform designed to monitor user activity, detect suspicious behavior, calculate risk scores, and generate fraud alerts in real time.

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication
* bcryptjs
* Git & GitHub

## Features Implemented

### Authentication

* User Registration
* User Login
* Password Hashing using bcrypt
* JWT Token Generation
* JWT Token Verification

### Authorization

* Role-Based Access Control (RBAC)
* USER and ADMIN roles
* Protected Routes

### Audit Logging

* Automatic API request logging
* Stores:

  * User ID
  * Endpoint
  * HTTP Method
  * IP Address
  * Timestamp

### Fraud Detection Engine (Phase 1)

* Failed Login Monitoring
* Failed Login Counter
* Dynamic Risk Score Tracking

### Alert Management

* Automatic alert generation for suspicious activity
* High-severity alerts for multiple failed login attempts
* Alert persistence in MongoDB Atlas

## Database Collections

### users

Stores user information, authentication data, risk scores, and account status.

### apilogs

Stores audit logs for all incoming API requests.

### alerts

Stores fraud alerts generated by the detection engine.

## Current Fraud Rules

### Multiple Failed Login Detection

Rule:

* After 5 failed login attempts:

  * Risk score increases
  * Fraud alert is generated

Example Alert:

{
"alertType": "MULTIPLE_FAILED_LOGINS",
"severity": "HIGH",
"status": "OPEN"
}
